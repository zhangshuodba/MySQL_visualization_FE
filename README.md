## 当前版本：v1.0.0

`注：基于iView-Admin模板开发，去除了所有拓展功能，只使用了基础框架。`

## Install
```bush
npm install
```
## Run
### Development
```bush
npm run dev
```
### Production(Build)
```bush
npm run build
```

## 文件结构
```shell
.
├── build  项目构建配置
└── src
    ├── images  图片文件
    ├── router  路由配置
    ├── store  状态管理
    ├── styles  样式文件
    ├── template  模板文件
    ├── vendors  公共库文件
    └── views
        ├── error-page  错误页面
        ├── home  首页
        ├── main-components  主组件
        ├── mysql  数据库状态展示组件
        ├── page  一级目录页面
```
